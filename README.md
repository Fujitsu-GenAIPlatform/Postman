# 初めに

Postman API プラットフォームを使って、Genearative AI Platform ( 以下、GAP ) の API を利用するためのサンプルになります。

## Postman API プラットフォームとは
[Postman API プラットフォーム](https://www.postman.com/jp/api-platform/)とは、チームがAPIを効果的に構築、管理、公開、利用できるように支援する、統合されたツールとプロセスを備えたソフトウェアシステムです。APIプラットフォームは、API利用者との直接的な連携も可能にし、既存のワークフローを補完・強化します。
